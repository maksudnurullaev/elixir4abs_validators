Необходимо реализовать пример генерацию QR кодов с примерами для разных продавцов, клиентов банка. 
Модуль генерирует уникальный идентификатор транзакции.
Генератор QR упаковывает данные (название магазина, счет, сумму и TTL) в графический файл.
Предполагается что покупатель сканирует код через любое банковское приложение, и деньги уходят напрямую со счета на счет.
